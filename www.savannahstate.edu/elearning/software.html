Software for Online Classes

Savannah State University uses the leading trends in technology available to online schools to give students an optimal online learning experience.

 

Software Resource & Services

Offering the University System of Georgia's faculty, staff and students affordable, brand name software and services for their work or personal computers at: http://www.srs.usg.edu/.

Web Browsers

:: Microsoft® Internet Explorer
Microsoft's Internet Explorer offers students the ability to use the web with enhanced security features and can be installed as an upgrade to existing versions of Internet Explorer.
:: Mozilla Firefox Browser
The award-winning Web browser is better than ever. Faster, safer, and easy to use, Firefox delivers a better Web experience.
:: Google Chrome
A browser that combines a minimal design with new technology designed to improve the user experience.
:: Apple Safari
Safari for Mac and PC puts the emphasis on browsing not the browser. Innovative features make your experience on the web better than it ever was.
:: Respondus LockDown Browser
Respondus LockDown Browser is a secure browser for taking quizzes and exams in Blackboard. If your course requires the use of Respondus LockDown browser, you will not able to take a quiz or exam with a standard web browser.

Respondus LockDown Browser should only be used for taking Blackboard quizzes or exams. It should not be used in other areas of Blackboard.

NOTE: In order to use Respondus LockDown browser, you must have Internet Explorer (PC) or Safari (Macintosh) installed on your computer and set as your default browser.

Browser Plug-ins

:: Java Plugin Console
Java lets you enjoy the best that the digital world has to offer, including games and puzzles,
sports, chat, email, the arts, financial tools and much more. You will also need to have
Java installed in order for certain areas of Blackboard to function properly.
:: Adobe Flash Player
Freedownload to view powerful video, multimedia, and application development features.
Many of our distance learning courses incorporate in a great deal of flash tutorials and
animations for a much more involved interactive learning experience.

Note: please UNCHECK the box "Also Install Yahoo Toolbar" (this is not required for operation of Flash Player and may cause Blackboard issues)
:: Adobe Shockwave Player
Some courses use interactive media which utilize powerful video, multimedia, and application development features.
:: Microsoft DirectX® End-User Runtime
Microsoft DirectX® is a group of technologies designed to make Windows-based computers an ideal platform for running and displaying applications rich in multimedia elements such as full-color graphics, video, 3D animation, and rich audio.

Free Word Processing Software

:: AbiWord
A free word processing program similar to Microsoft® Word. It is suitable for a wide variety of word processing tasks.
:: Google Docs Create and share documents on the web and access them from any computer or smart phone. The familiar desktop feel makes collaborative editing easy.
:: OpenOffice
The leading free open-source office software suite for word processing, spreadsheets, presentations, graphics, databases and more.

Other Free Programs

 

:: Adobe® Acrobat® Reader®
Free software that lets you view and print Adobe Portable Document Format (PDF) files on all major computer platforms.
:: Apple iTunes Player
Free software that lets you organize and play everything on your Mac or PC. Then sync it to your iPod and bring it along. Anywhere.
:: Apple QuickTime® Player
Cutting-edge digital media software for both Mac and Windows-based computers delivers unparalleled quality for creating, playing and streaming audio and video content over the Internet.
:: Microsoft DirectX® End-User Runtime
Microsoft DirectX® is a group of technologies designed to make Windows-based computers an ideal platform for running and displaying applications rich in multimedia elements such as full-color graphics, video, 3D animation, and rich audio.
:: Microsoft® Windows & Office Updates
It is important to keep your computer current with all Microsoft updates for computer efficiency and security. Download free updates and patches available at Microsoft.
:: Real™ Player
The Real Basic Player gives you access to RealAudio, RealVideo, live events and more. The free player is called "Basic." RealOne SuperPass is an application for purchase by the company.
